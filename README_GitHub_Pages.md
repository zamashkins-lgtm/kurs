
# Публикация на GitHub Pages

1. Создайте публичный репозиторий `port-monitor`.
2. Загрузите в корень репозитория все файлы из этой папки.
3. В GitHub откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите:
   - Source: `Deploy from a branch`
   - Branch: `main` / `(root)`
5. Сохраните настройки.

Через 1–2 минуты сайт будет доступен по адресу:

`https://ВАШ_ЛОГИН.github.io/port-monitor/`
